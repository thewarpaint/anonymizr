# Anonymizr

Remove personal information from social networks and other websites before sharing a screenshot.

## Supported sites

+ GitHub
+ Reddit
+ Twitter

### To be supported

Click on the site name to vote for it!

+ [Facebook](https://github.com/thewarpaint/anonymizr/issues/1)
